# C-Condicionais-Lista
# 💻 Exercícios: Introdução a Sistemas de Informação

Repositório destinado à resolução da **Lista de Exercícios 2** da disciplina de **Introdução a Sistemas de Informação**.O foco deste módulo é a implementação de **Estruturas Condicionais** (logística de decisão). [cite: 1, 2]

## 🚀 Sobre o Projeto

[cite_start]Este projeto contém algoritmos desenvolvidos para resolver problemas de lógica computacional, simulando cenários reais de sistemas corporativos, financeiros e de saúde. [cite: 2, 21, 25, 40]

### 🛠️ Conceitos Aplicados
* [cite_start]**Estruturas de Decisão:** Uso de `if`, `else if`, `else` e `switch/case`. [cite: 3, 10, 13]
* [cite_start]**Lógica de Negócios:** Automação de reajustes salariais e cálculos de impostos. [cite: 16, 29, 31, 82]
* [cite_start]**Sistemas Financeiros:** Cálculo de crédito especial e rendimento de investimentos. [cite: 21, 22, 40, 41]
* [cite_start]**Validação:** Verificação de senhas, maioridade e categorias esportivas. [cite: 46, 48, 53, 54]

## 📝 Exemplo de Regra de Negócio

[cite_start]Um dos algoritmos implementados (Exercício 25) calcula a gratificação de Natal baseada em horas líquidas utilizando a fórmula: [cite: 90, 93]

[cite_start]$$H = (horas\_extras) - \frac{2}{3} * (horas\_falta)$$ [cite: 93]

[cite_start]O valor do prêmio é definido conforme a faixa de pontuação obtida em $H$. [cite: 94]
