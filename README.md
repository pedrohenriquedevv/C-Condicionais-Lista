# C-Condicionais-Lista
# 💻 Exercícios: Introdução a Sistemas de Informação

Repositório destinado à resolução da **Lista de Exercícios 2** da disciplina de **Introdução a Sistemas de Informação**.O foco deste módulo é a implementação de **Estruturas Condicionais** (logística de decisão).
(SEM USO DE IA!!!)
## 🚀 Sobre o Projeto

Este projeto contém algoritmos desenvolvidos para resolver problemas de lógica computacional, simulando cenários reais de sistemas corporativos, financeiros e de saúde. 

### 🛠️ Conceitos Aplicados
* **Estruturas de Decisão:** Uso de `if`, `else if`, `else` e `switch/case`.
* **Lógica de Negócios:** Automação de reajustes salariais e cálculos de impostos.
* **Sistemas Financeiros:** Cálculo de crédito especial e rendimento de investimentos.
* **Validação:** Verificação de senhas, maioridade e categorias esportivas.

## 📂 Resumo da Lista de Exercícios

A lista é composta por 25 desafios práticos:

| Exercício | Categoria | Descrição |
| :--- | :--- | :--- |
| **01 - 04** | Acadêmico | Médias aritméticas, aprovação escolar e comparação de valores. |
| **05 - 06** | Operações | Menu interativo de cálculos matemáticos com tratamento de erros. |
| **07 - 14** | RH & Finanças | Reajustes de salário, cálculo de crédito bancário e impostos de fábrica. |
| **15 - 17** | Segurança | Escolha de investimentos e validação de senha de acesso. |
| **18 - 22** | Saúde/Social | Verificação de maioridade, peso ideal e grupos de risco. |
| **23 - 25** | Corporativo | Gestão de notas fiscais, taxas sobre produtos e bônus de Natal. |

## 📝 Exemplo de Regra de Negócio

Um dos algoritmos implementados (Exercício 25) calcula a gratificação de Natal baseada em horas líquidas utilizando a fórmula: 

$$H = (horas\_extras) - \frac{2}{3} * (horas\_falta)$$ 

O valor do prêmio é definido conforme a faixa de pontuação obtida em $H$.
